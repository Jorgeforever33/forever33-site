Aclaración fiscal

Comprobantes internos no fiscales

F33 no incluye facturación fiscal en esta etapa. El sistema puede generar comprobantes comerciales internos no fiscales, pero no reemplaza factura fiscal, AFIP, CAE ni facturación electrónica.

Tampoco incluye instalador MSI automático, servicio Windows ni descarga pública autoservicio por ahora.

Preguntas frecuentes

Dudas comunes antes de probar el sistema

Un resumen rápido para entender cómo se instala, cómo se usa, qué incluye y qué no incluye F33 POS.

¿Funciona sin internet?

Sí. El sistema está pensado para trabajar de forma local en la PC del comercio, ferretería o distribuidora. Algunas funciones futuras podrían requerir internet, pero el uso principal del POS no depende de estar conectado.

¿Sirve también para distribuidoras o mayoristas?

Sí. Esta versión está pensada para ferreterías, distribuidoras y comercios con muchos productos, listas de precios, clientes frecuentes o mayoristas, stock y pedidos comerciales. El piloto puede ajustarse según el uso real del negocio.

¿Se puede usar desde celulares?

Sí. La idea es que el sistema corra en la PC principal y que otros dispositivos, como celulares o tablets, puedan acceder dentro de la misma red local del negocio.

¿Cómo se activa la licencia?

La activación se gestiona manualmente. Consultás por WhatsApp, acordamos una propuesta para tu comercio y se genera una licencia local para la instalación del cliente.

¿Se puede instalar ahora?

Sí, como demo o piloto local asistido. La versión actual requiere instalación acompañada y configuración inicial; la descarga pública autoservicio queda para una etapa futura.

¿Incluye facturación fiscal?

No. F33 no incluye facturación fiscal en esta etapa. Puede generar comprobantes comerciales internos no fiscales, pero no reemplaza factura fiscal, AFIP, CAE ni facturación electrónica.

¿Se puede adaptar al presupuesto?

Sí. Revisamos el caso por WhatsApp, buscamos una propuesta acorde al presupuesto y podemos sumar funciones si el comercio realmente las necesita.

¿Incluye soporte?

Sí. El contacto principal será por WhatsApp para consultas, instalación, activación y acompañamiento inicial.
