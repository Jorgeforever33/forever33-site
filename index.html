Preguntas frecuentes

Dudas comunes antes de probar el sistema

Un resumen rápido para entender cómo se instala, cómo se usa y cómo se activa el POS.

¿Funciona sin internet?

Sí. El sistema está pensado para trabajar de forma local en la PC del comercio. Algunas funciones futuras podrían requerir internet, pero el uso principal del POS no depende de estar conectado.

¿Se puede usar desde celulares?

Sí. La idea es que el sistema corra en la PC principal y que otros dispositivos, como celulares o tablets, puedan acceder dentro de la misma red local del negocio.

¿Cómo se activa la clave mensual?

La activación se gestiona manualmente. Consultás por WhatsApp, elegís el plan disponible y recibís una clave para habilitar el uso durante el período contratado.

¿Cuándo estará disponible la descarga?

La descarga pública se habilitará cuando el producto esté listo para entregar. Mientras tanto, podés consultar por demo, avances y disponibilidad.

¿Incluye soporte?

Sí. El contacto principal será por WhatsApp para consultas, instalación, activación y acompañamiento inicial.
