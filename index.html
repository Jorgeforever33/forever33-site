Instalación local

Instalación local asistida en Windows

F33 POS cuenta con un instalador unificado asistido para Windows. Desde el instalador podés elegir entre una Demo con productos cargados o una Base limpia para uso real.

Node.js es un requisito externo: el instalador no lo incluye ni lo descarga. Si falta, F33 POS muestra el aviso y guía al usuario para instalarlo desde la página oficial.

Preguntas frecuentes

Dudas comunes antes de probar el sistema

Un resumen rápido para entender cómo se instala, cómo se usa, qué incluye y qué no incluye F33 POS.

¿Funciona sin internet?

Sí. El sistema está pensado para trabajar de forma local en la PC principal del comercio o negocio. Algunas funciones futuras podrían requerir internet, pero el uso principal del POS no depende de estar conectado.

¿Sirve solo para ferreterías?

No. F33 POS es una base general para comercios. Puede adaptarse a distintos rubros como ferreterías, kioscos, distribuidoras u otros negocios personalizados.

¿Qué diferencia hay entre Demo y Base?

La Demo trae productos cargados para probar y mostrar el sistema. La Base viene limpia para usar con datos reales del negocio.

¿Se puede usar desde celulares?

Sí. La idea es que el sistema corra en la PC principal y que otros dispositivos, como celulares o tablets, puedan acceder dentro de la misma red local del negocio.

¿Cómo se activa?

Se activa con un código F33-ACT generado para la instalación del cliente. Cada instalación genera un InstallationID y soporte entrega la clave correspondiente.

¿Instala Node.js?

No. Node.js es un requisito externo. Si falta, F33 POS muestra el aviso y permite ir a la página oficial para instalarlo.

¿Incluye facturación fiscal?

F33 POS no incluye facturación fiscal, AFIP, CAE ni factura electrónica en esta etapa. El sistema puede generar comprobantes comerciales internos no fiscales, pero no reemplaza la facturación fiscal.

¿Se puede adaptar al presupuesto?

Sí. Revisamos el caso por WhatsApp, buscamos una propuesta acorde al presupuesto y podemos sumar funciones si el comercio realmente las necesita.

¿Incluye soporte?

Sí. El contacto principal será por WhatsApp para consultas, instalación, activación y acompañamiento inicial.
